# Builg-Front

## Setup

Get the code by either cloning this repository using git

```
git clone https://github.com/ArturRushanyan/Builg-Front
```

Mote to the repository directory

Create a `.env` file in the project root and copy the content from `.env.example`:

```bash
  cp .env.example .env
```

Add the required configurations to the `.env` file.

Run the command to install dependency:

```
npm install
```

## Run application

Then start the app in development mode with command:

```
npm run dev
```
